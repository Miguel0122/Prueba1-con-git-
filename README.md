# Proyecto de Práctica con Git

Este repositorio fue creado con el objetivo de practicar las operaciones básicas de Git utilizando GitHub Web.  
Aquí se aplican conceptos como creación de repositorios, edición de archivos, commits y visualización del historial.

  Objetivos del ejercicio
- Crear un repositorio en GitHub  
- Crear y editar un archivo README.md  
- Realizar commits  
- Modificar archivos y registrar los cambios  
- Explorar el historial de commits

 Este archivo README fue actualizado como parte de las prácticas del curso.
